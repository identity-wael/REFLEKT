# Calm Mode

REFLEKT's **Calm Mode** is designed to help you start your day centered and mindful. Through guided breathing exercises and calming visualizations, Calm Mode transforms your smart mirror into a wellness companion that reduces anxiety and promotes mental well-being.

## Overview

Calm Mode provides a serene, distraction-free interface focused on mindfulness and emotional wellness. When activated, the mirror displays soothing visuals, breathing guides, and ambient feedback to help you achieve a calm, centered state before beginning your day.

### Key Features

- **Guided Breathing Exercises**: Real-time visual breathing guides with customizable patterns
- **Mindfulness Moments**: Short meditation sessions tailored to your available time
- **Mood Detection Integration**: Adapts exercises based on detected emotional state
- **Ambient Soundscapes**: Optional calming audio to enhance relaxation
- **Progress Tracking**: Monitor your mindfulness practice over time

## How It Works

### Activation

Calm Mode can be activated through:

1. **Voice Command**: "Hey REFLEKT, start calm mode"
2. **Touch Gesture**: Swipe up from the bottom of the mirror
3. **Automatic Trigger**: Based on detected stress levels or time of day
4. **Scheduled Sessions**: Set recurring mindfulness times

### Breathing Exercises

REFLEKT offers several breathing patterns scientifically proven to reduce anxiety:

#### Box Breathing (4-4-4-4)
```
Inhale:  4 seconds → visual circle expands
Hold:    4 seconds → circle pauses at full size
Exhale:  4 seconds → circle contracts
Hold:    4 seconds → circle pauses at minimum
```

**Best for**: Stress reduction, focus improvement

#### 4-7-8 Breathing
```
Inhale:  4 seconds through nose
Hold:    7 seconds
Exhale:  8 seconds through mouth
```

**Best for**: Sleep preparation, anxiety relief

#### Resonant Breathing (5-5)
```
Inhale:  5 seconds
Exhale:  5 seconds
```

**Best for**: Heart rate variability, emotional balance

### Visual Feedback

The mirror provides real-time visual cues:

- **Expanding/Contracting Circle**: Guides breathing rhythm
- **Color Transitions**: Calming gradient shifts (blues, greens, purples)
- **Ambient Light**: Adjusts mirror brightness for comfort
- **Text Prompts**: Optional on-screen guidance

- **Anxious State**: Recommends 4-7-8 breathing technique

## Troubleshooting

### Common Issues

**Q: The breathing circle is too fast/slow**
- A: Adjust timing in Settings → Calm Mode → Breathing Pattern

**Q: I can't hear the audio**
- A: Check mirror volume settings and ensure audio is enabled

**Q: Mood detection suggests Calm Mode too often**
- A: Calibrate sensitivity in Settings → Mood Sensing

**Q: Session interrupted unexpectedly**
- A: Enable "Do Not Disturb" mode during Calm sessions
