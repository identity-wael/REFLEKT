## Contextual Intelligence

### Adaptive Information Display

Active Mode adjusts based on:

**Time of Day**:
- **Early Morning (5-7 AM)**: Gentle wake-up info, minimal brightness
- **Morning Rush (7-9 AM)**: Full briefing, urgent items highlighted
- **Midday Check-in**: Afternoon schedule preview
- **Evening**: Next day preparation

**Day of Week**:
- **Monday**: Week planning, longer briefings
- **Friday**: Weekend preview, wrap-up tasks
- **Weekend**: Relaxed pace, personal activities focus

**Detected Stress Level**:
- **Calm**: Full information display
- **Rushed**: Condensed view, key items only
- **Stressed**: Calming reminder + essential info only
